# Interface: ContextualGroundingFilter

Interface to define a Contextual Grounding Filter.

## Properties

### threshold

> `readonly` **threshold**: `number`

The threshold for the contextual grounding filter.
- `0` (blocks nothing)
- `0.99` (blocks almost everything)
