## Enumeration Members

### ASSISTANT

> **ASSISTANT**: `"assistant"`

the context set by the system.

***

### USER

> **USER**: `"user"`

This role represents the human user in the conversation. Inputs from the
user guide  the conversation and prompt responses from the assistant.
